# VLC Open Source Project

## Description
This project is based on VLC Media Player, which is a free and open-source software. The goal of this project is to understand how open-source software works and how it can be used in real life.

## Software Used
VLC Media Player

## Scripts Included

1. Script 1 - System Info
Displays basic system information like user, date, and system details.

2. Script 2 - VLC Info
Provides basic information about VLC and its features.

3. Script 3 - Directory Check
Checks if certain directories exist in the system.

4. Script 4 - Log Analyzer
Creates a log file and counts error lines.

5. Script 5 - User Input Script
Takes user input and generates a simple output.

## How to Run

Use any Linux system or online bash compiler.

Example:
bash script1.sh
